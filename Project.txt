# Cars - Vue.js SPA project

## What is the main purpose of the project

To create and see information about cars, in list



### Logged out user - can only see guest-home page and Login, Register



### Logged in user - has full access to the project

- Can create Cars - model, price, picture of the car and others.
- Can see the full list of the cars.They are being stored in cloud database - Kinvey.

- Can see the full details of the car.
